# 🚀 ENGCE301: Software Design and Development Lab

คลังเก็บรวบรวมปฏิบัติการและโปรเจกต์รายวิชา **ENGCE301 การออกแบบและพัฒนาซอฟต์แวร์** ประจำปีการศึกษา 2568

## 📌 ภาพรวมโปรเจกต์
Repository นี้ประกอบด้วยผลงานปฏิบัติการรายสัปดาห์ ตั้งแต่พื้นฐานการพัฒนาเว็บ ไปจนถึงสถาปัตยกรรมซอฟต์แวร์ขั้นสูงและการปรับแต่งประสิทธิภาพ

---

## 📅 สรุปกิจกรรมรายสัปดาห์ (Weekly Labs)

### [Week 1: Introduction to Software Engineering](./week1)
- การเตรียมสภาพแวดล้อมการพัฒนา (VS Code, Node.js, Git)
- พัฒนาหน้า "Hello World" Landing Page ด้วยเทคนิค Glassmorphism
- พื้นฐาน SDLC และ Agile Methodology

### [Week 2: Version Control & Collaborative Development](./week2)
- การใช้ Git Feature Branch Workflow ในการทำงานจริง
- การจัดการ Pull Requests และกระบวนการ Code Review
- คำสั่ง Git ขั้นพื้นฐานและขั้นสูงสำหรับการทำงานร่วมกัน

### [Week 3: Requirements Analysis & User Stories](./week3)
- การวิเคราะห์ความต้องการ (Functional & Non-functional Requirements)
- การเขียน User Stories และ Criteria การยอมรับ (Acceptance Criteria)
- การระดมสมองเพื่อแก้ปัญหาทางธุรกิจด้วยเทคโนโลยี

### [Week 4: Web Portfolio Development](./week4)
- พัฒนาหน้าเว็บพอร์ตโฟลิโอเบื้องต้นด้วย Semantic HTML5
- ออกแบบ UI ที่ทันสมัยและรองรับการแสดงผลบนมือถือ (Responsive Design)

### [Week 5: DOM Manipulation & Game Logic](./week5)
- พื้นฐาน JavaScript และการควบคุมหน้าเว็บผ่าน DOM Manipulation
- พัฒนาเกม Tic-Tac-Toe พร้อมระบบตรวจสอบเงื่อนไขการชนะ

### [Week 6: RESTful API with Express.js](./week6)
- เริ่มต้นพัฒนาฝั่ง Backend ด้วย Node.js และ Express
- การพัฒนา RESTful API เบื้องต้นและการจัดการ Routing

### [Week 7: Layered Architecture (Refactored)](./week7)
- การทำ Refactoring โค้ดให้เป็นระบบ Layered Architecture (Controller, Service, Database Layer)
- เพิ่มความสามารถในการจัดการข้อมูลผ่าน SQLite อย่างมืออาชีพ

### [Week 8: Backend Deep Dive](./week8)
- การพัฒนา API ที่รับ-ส่งข้อมูลรูปแบบ JSON อย่างมีประสิทธิภาพ
- การจัดการ Middleware และ Error Handling ในระดับลึก

### [Week 9: Database Integration (SQLite)](./week9)
- การเชื่อมต่อฐานข้อมูล SQL แบบถาวร (Persistent Storage)
- การเขียนคำสั่ง SQL เพื่อจัดการข้อมูลในระบบอย่างเป็นระบบ

### [Week 12: Security Architecture & JWT](./week12)
- การออกแบบสถาปัตยกรรมความปลอดภัยสำหรับ Microservices
- การเขียนเอกสารตัดสินใจทางสถาปัตยกรรม (ADR) และแผนการรวมระบบ

## 🏗️ ปฏิบัติการโครงการขั้นสูง (Advanced Labs)

### [Lab: JWT & Microservices](./lab-jwt-microservices)
- การทำระบบยืนยันตัวตนแบบ Decentralized ด้วย JWT (JSON Web Token)
- การใช้ Nginx เป็น API Gateway สำหรับควบคุม Microservices ย่อยๆ

### [Lab: N-Tier & Containerization](./lab-ntier-docker)
- การบรรจุซอฟต์แวร์ลงคอนเทนเนอร์ด้วย Docker และ Docker Compose
- การนำแอปพลิเคชันขึ้นใช้งานจริงบนระบบ Cloud (Railway)

### [Lab: Scaling & Performance](./lab-ntier-redis-loadbalance)
- การเพิ่มประสิทธิภาพระบบด้วย Redis Caching
- การทำ High Availability ด้วย Nginx Load Balancing และ Failover

## 🛠️ เครื่องมือหลักที่ใช้
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), React.js
- **Backend:** Node.js, Express.js
- **Database:** SQLite, Redis
- **DevOps/Infra:** Git, Docker, Nginx, Railway
