# ปฏิบัติการสัปดาห์ที่ 4: การพัฒนาเว็บพอร์ตโฟลิโอเบื้องต้น (Web Portfolio)

## 🎯 ภาพรวมรายวิชา
- **วิชา:** ENGCE301 การออกแบบและพัฒนาซอฟต์แวร์
- **หัวข้อ:** การสร้างโครงสร้างและออกแบบหน้าเว็บด้วย HTML5 และ CSS3

## 📚 สิ่งที่ได้เรียนรู้
ในสัปดาห์นี้เราเปลี่ยนจากการออกแบบโมเดล (UML) มาเน้นที่การนำเอาความต้องการของผู้ใช้ (Requirements) มาสร้างเป็นส่วนติดต่อผู้ใช้ (UI) จริงๆ ผ่านโปรเจกต์ **Web Portfolio**

### 1. Semantic HTML
การเลือกใช้ Element ที่มีความหมาย เช่น `<header>`, `<nav>`, `<main>`, `<section>`, และ `<footer>` เพื่อช่วยให้โครงสร้างเว็บชัดเจนและเป็นมิตรกับ Search Engine (SEO)

### 2. Modern CSS Design
- **Dark Mode Concept:** การใช้โทนสีมืดที่ดูพรีเมียมและสบายตา
- **Glassmorphism:** การทำเอฟเฟกต์กระจกฝ้าด้วย `backdrop-filter: blur()` และความโปร่งใสแบบกึ่งโปร่งแสง
- **Responsive Design:** การออกแบบให้รองรับทั้งหน้าจอคอมพิวเตอร์และมือถือผ่าน Media Queries

## 📂 โครงสร้างโปรเจกต์
- `index.html`: โครงสร้างหลักของหน้าเว็บ (About, Education, Hobbies, Contact)
- `style.css`: การกำหนดรูปแบบและสไตล์ที่เน้นความทันสมัย

## 🚀 วิธีการเข้าชม
เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ใดก็ได้ที่คุณเลือก
