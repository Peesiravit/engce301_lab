# Lab Reflection - Week 8

## สิ่งที่ได้เรียนรู้ (What I Learned)
สัปดาห์นี้เป็นก้าวแรกเข้าสู่โลกของ Backend Development ซึ่งแตกต่างจากการเขียนหน้าบ้าน (Frontend) อย่างสิ้นเชิง

### ประเด็นสำคัญ:
1.  **Request-Response Cycle:** เข้าใจกลไกการสื่อสารระหว่างไคลเอนต์และเซิร์ฟเวอร์ผ่าน HTTP Methods (GET, POST)
2.  **Express.js Efficiency:** ความประทับใจในความง่ายของ Express ที่ช่วยให้เราสร้างเซิร์ฟเวอร์ขึ้นมาได้ภายในไม่กี่บรรทัด
3.  **Data Management:** แม้จะยังไม่ได้ใช้ฐานข้อมูลจริง (ใช้ in-memory array) แต่ก็ได้เห็นภาพว่าเซิร์ฟเวอร์จัดการข้อมูลอย่างไรก่อนที่จะส่งกลับไปให้หน้าบ้าน
4.  **Error Handling:** เรียนรู้การใช้ HTTP Status Codes (เช่น 200 OK, 201 Created, 400 Bad Request, 404 Not Found) เพื่อบอกสถานะการทำงานให้ผู้ใช้งานทราบ

### ความท้าทายที่พบ:
- การจัดลำดับ Middleware และความสำคัญของการใช้ `express.json()` เพื่อให้อ่านข้อมูลจาก Request Body ได้
- การทำความเข้าใจเรื่อง Parameters (เช่น `:id` ใน URL) และวิธีการดึงค่าออกมาใช้งาน

## เป้าหมายในอนาคต
