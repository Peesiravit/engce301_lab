# Lab Reflection - Week 9

## สิ่งที่ได้เรียนรู้ (What I Learned)
สัปดาห์นี้เป็นจุดเปลี่ยนสำคัญที่ทำให้แอปพลิเคชันมีความเป็นมืออาชีพมากขึ้น เพราะข้อมูลสามารถจัดเก็บได้อย่างถาวร

### ประเด็นสำคัญ:
1.  **Persistence (การคงอยู่ของข้อมูล):** ผมได้เห็นความแตกต่างระหว่างการเก็บข้อมูลในตัวแปรและการใช้ฐานข้อมูลจริง เมื่อปิด-เปิดเซิร์ฟเวอร์ใหม่ ข้อมูลที่บันทึกไว้ใน SQLite ก็ยังคงอยู่
2.  **SQL Basics:** ได้ฝึกใช้คำสั่ง SQL พื้นฐาน (SELECT, INSERT, DELETE) ซึ่งเป็นทักษะพื้นฐานที่สำคัญมากสำหรับนักพัฒนา Software
3.  **Asynchronous Database Calls:** เรียนรู้การจัดการ Callback ใน Node.js เมื่อต้องทำงานกับฐานข้อมูล เนื่องจากเราไม่ทราบแน่นอนว่าผลลัพธ์จะกลับมาเมื่อไหร่
4.  **Schema Design:** เข้าใจความสำคัญของการวางโครงสร้างตาราง (Columns, Data Types, Primary Key) ให้เหมาะสมกับข้อมูลที่เราต้องการจัดเก็บ

### ความท้าทายที่พบ:
- การจัดการข้อผิดพลาด (Error Handling) จากฐานข้อมูล เช่น กรณีที่คำสั่ง SQL ผิดพลาด หรือหาไฟล์ฐานข้อมูลไม่เจอ
- การจัดการกับลำดับการทำงาน (Asynchronous sequence) เพื่อให้แน่ใจว่าตารางถูกสร้างก่อนที่ API จะเริ่มทำงาน
