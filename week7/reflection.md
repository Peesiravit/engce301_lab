# Lab Reflection - Week 7

## สิ่งที่ได้เรียนรู้ (What I Learned)
สัปดาห์นี้เป็นการยกระดับจากการเขียนโค้ดเพื่อออกแบบ (LLD) มาเป็นการสร้างแอปพลิเคชันที่ใช้งานได้จริง (Functional App)

### ประเด็นสำคัญ:
1.  **Stateful Development:** เข้าใจลึกซึ้งขึ้นเกี่ยวกับการจัดการ State ในกรณีที่ข้อมูลมีความสัมพันธ์กันและมีจำนวนมาก (Array of Tasks)
2.  **Hooks (The Logic):** การใช้ `useEffect` ร่วมกับ `localStorage` ทำให้ผมเข้าใจว่าเราสามารถเชื่อมต่อหน้าบ้าน (Frontend) เข้ากับกลไกการเก็บข้อมูลพื้นฐานได้อย่างไร
3.  **User Experience (UX):** การเพิ่มฟีเจอร์อย่างการขีดฆ่างานที่ทำเสร็จแล้ว (Text-decoration line-through) หรือการเลือกสีที่สื่อถึงสถานะ ช่วยให้แอปดูเป็นมิตรกับผู้ใช้มากขึ้น
4.  **Component Strategy:** การรักษาโครงสร้างแอปให้แยกส่วนกันได้ดี (แม้จะเขียนรวมในไฟล์เดโมไฟล์เดียว) ช่วยให้เห็นความสำคัญของการวางผังที่วาดไว้ในสัปดาห์ที่ 6

### ความท้าทายที่พบ:
- การจัดการกับ Side Effects ของ React (การที่แอปวาดซ้ำเมื่อข้อมูลเปลี่ยน) ทำให้ต้องระมัดระวังเรื่องการอัปเดต State ที่ซ้อนทับกัน
- การทำระบบ persistence ด้วย localStorage ที่ต้องจัดการกับการแปลงข้อมูล (JSON.parse/stringify) ให้ถูกต้องเสมอ
