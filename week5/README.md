# ปฏิบัติการสัปดาห์ที่ 5: การจัดการ DOM และตรรกะโปรแกรม (DOM Manipulation & Game Logic)

## 🎯 ภาพรวมรายวิชา
- **วิชา:** ENGCE301 การออกแบบและพัฒนาซอฟต์แวร์
- **หัวข้อ:** การใช้ JavaScript ในการควบคุมโครงสร้างหน้าเว็บ (DOM) และการสร้างระบบ Logic

## 📚 สิ่งที่ได้เรียนรู้
ในสัปดาห์นี้เราเน้นการใช้ JavaScript เพื่อเปลี่ยนให้หน้าเว็บที่หยุดนิ่ง (Static) กลายเป็นเว็บที่มีการโต้ตอบ (Interactive) ผ่านกิจกรรมดังนี้:

### 1. DOM Challenges (1-4)
- **Challenge 1:** การเข้าถึง Element และการเปลี่ยนสไตล์ (สี, ขนาดตัวอักษร)
- **Challenge 2:** การใช้ `querySelector` ในการเลือก Element ที่มีความซับซ้อน (เช่น :nth-child)
- **Challenge 3:** การใช้งาน `addEventListener` เพื่อตรวจจับการคลิกปุ่ม
- **Challenge 4:** การควบคุมพฤติกรรมดั้งเดิมของ Browser ด้วย `preventDefault()`

### 2. Tic-Tac-Toe Game
- **State Management:** การเก็บสถานะของกระดานเกมในรูปแบบ Array
- **Algorithm:** การตรวจสอบเงื่อนไขการชนะ (Winning Conditions) ทั้ง 8 รูปแบบ
- **UI Interaction:** การอัปเดตหน้าแสดงผลตามรอบของผู้เล่นและการประกาศผู้ชนะแบบ Real-time

## 📂 โครงสร้างโปรเจกต์
- `challenges/`: แบบฝึกหัดพื้นฐาน 4 ช่องทาง
- `tic-tac-toe/`: โปรเจกต์เกมเต็มรูปแบบที่มีการแยก HTML, CSS, และ JavaScript

## 🚀 วิธีการเข้าชม
เปิดไฟล์ `index.html` ในโฟลเดอร์ที่ต้องการด้วยเว็บเบราว์เซอร์
